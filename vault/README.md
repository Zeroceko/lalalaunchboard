# Vault

This folder is designed to work well as an Obsidian vault.

## Purpose

Use this folder for:

- decision records
- architecture notes
- dev logs
- backlog notes
- loose working memory

## Rule

`specs/` is the official source of truth.

Do not mirror or fork the spec files here. Instead, summarize, reference, and link back to them.

## Suggested Usage

- `00-inbox/` - quick captures
- `01-decisions/` - decision records
- `02-architecture/` - system notes
- `03-product/` - product thinking and tradeoffs
- `04-devlog/` - dated progress logs
- `05-backlog/` - unstructured follow-up items
